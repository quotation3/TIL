# Git basic command

> git 명령어 기초 정리



### init

- 초기에 폴더를 깃으로 관리하겠다라고 선언하는 명령어
- 현재 폴더에 `.git/` 폴더가 생성된다.



### add

- 중간저장, 임시저장
- `git add.` 명령어를 통해 전체 파일을 한번에 add 할수 있다.



### commit

- add 되어있는 파일들을 하나의 버전으로 묶어주는 명령어
- `-m` 옵션으로 커밋메세지를 입력 할 수 있다.



### push

- 로컬에 있는 커밋기록을 원격저장소로 업로드 한다.



### clone

- 원격 저장소에 있는 프로젝트를 복제하는 명령어

```shell
$ git clone <원격저장소 주소>
```



### pull

- 원격 저장소에서 최근 커밋을 불러오는 명령어
